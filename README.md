# 🏛️ Mesa Única: Sistema de Gestión de Expedientes

> **Transformando la interacción entre el ciudadano y el municipio.**

## 📋 Visión General

**Mesa Única** es una plataforma centralizada diseñada para modernizar y agilizar la gestión de trámites municipales. El sistema actúa como un nexo digital entre los contribuyentes y las diversas áreas técnicas del municipio, eliminando cuellos de botella en la comunicación y proporcionando transparencia absoluta sobre el estado de la documentación.

El objetivo principal es transformar el proceso administrativo en una experiencia fluida, donde la "Carpeta de Obra" actúa como el eje central de toda la actividad constructiva y legal del ciudadano.

---

## 👥 Actores del Ecosistema

| Actor | Responsabilidad Clave |
| :--- | :--- |
| **Contribuyente** | Gestionar sus trámites, cargar documentación y subsanar observaciones en tiempo real. |
| **Mesa de Entrada** | Recepción formal, vinculación administrativa con el sistema GDE y alta de expedientes. |
| **Sectores Técnicos** | Revisión especializada (Catastro, Planeamiento, Medio Ambiente, etc.) y visado de documentos. |

---

## 🚀 Flujo Operativo: El Concepto de "Madre e Hijo"

El sistema implementa una arquitectura de expedientes inteligente para evitar la redundancia de datos:

1.  **Expediente Madre (Carpeta de Obra):** Es el punto de partida obligatorio. Contiene la documentación base del inmueble y el propietario que es común a todos los trámites.
2.  **Expedientes Hijos (Trámites Específicos):** Certificados de Zonificación, Prefactibilidad o Impacto Ambiental se vinculan a la Carpeta de Obra. 
3.  **Validación Cruzada:** Los sectores técnicos pueden consultar la documentación base de la "Madre" mientras evalúan los requisitos específicos del "Hijo".

---

## ✨ Características Principales

*   **🔍 Transparencia Total:** El contribuyente conoce en todo momento quién tiene su documento y por qué ha sido observado.
*   **📂 Repositorio Centralizado:** Adiós a las copias infinitas. Los documentos se cargan una vez y sirven de referencia para múltiples sectores.
*   **💬 Comunicación Directa:** Canal de mensajería integrado por expediente para resolver dudas técnicas sin necesidad de traslados físicos.
*   **✅ Control de Estados:** Ciclo de vida claro para cada documento: *Pendiente, Cargado, En Revisión, Aprobado o Pendiente de Resubida*.
*   **📅 Historial de Auditoría:** Registro inalterable de cada acción realizada sobre un expediente para garantizar la seguridad jurídica.

---

## 🏗️ Alcance Administrativo

El sistema cubre la gestión de los siguientes hitos municipales:
*   **Obras Particulares:** Gestión integral de la Carpeta de Obra.
*   **Planeamiento:** Certificados de Zonificación y Prefactibilidad.
*   **Medio Ambiente:** Declaraciones de No Impacto Ambiental.
*   **Áreas de Apoyo:** Intervención de Catastro, Infraestructura y Obras Hídricas.
